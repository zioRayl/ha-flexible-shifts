# Flexible Shifts

Add-on Home Assistant per la gestione di turni non standard, pause, preset orari per utente, settimane di ferie e report annuali.

Ogni turno contiene un solo **Inizio turno**, una sola **Fine turno** e, facoltativamente, una pausa da sottrarre.

Consultare il README principale del repository per installazione, logica dei conteggi e formato CSV.


## Accesso utenti

La voce **Turni** è accessibile a tutti gli utenti autenticati di Home Assistant. La gestione dell’add-on (installazione, aggiornamento, avvio e configurazione) rimane invece riservata agli amministratori di Home Assistant.
